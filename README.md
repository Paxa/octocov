# octocov

![coverage](docs/coverage.svg)

`octocov` is a tool for collecting code coverage.

## Supported Coverage Report Format

### Go coverage

**Default path:** `coverage.out`

### LCOV

**Default path:** `coverage/lcov.info`

Support `SF` `DA` only

### SimpleCov

**Default path:** `coverage/.resultset.json`
